Backport of Blizzard's Stopwatch from TBC to 1.12 by Zerf
Version 1.0

URI: https://github.com/Zerf/Stopwatch

Command examples:
Show UI: /stopwatch or /timer
Reset UI position: /stopwatch reset-position
Start coundown timer for 1h 2min 3sec: /stopwatch 01:02:03
Start coundown timer for 50sec: /stopwatch 50
Start timer: /stopwatch play
Pause timer: /stopwatch pause
Reset timer: /stopwatch reset
